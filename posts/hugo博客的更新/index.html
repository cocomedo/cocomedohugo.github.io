** 关于如何上传自己的hugo博客新文章 **

如果你已经完整的经历过了hugo（雨果）博客的下载安装远程部署到GitHub上，我相信你大概对hugo博客的大致操作有所了解了，但是我也知道一定还有部分同学可能会忘记怎么上传和更新自己的hugo博文。

打开命令提示符界面（CMD）进入到 hugo博客文件目录下的public（这个文件夹主要是存放上传到GitHub上的页面样式和内容）

git init

git add .

注意 add 和 . 之间有个空格，不要漏了

git commit -m "****"

-m 后面的"" 内的内容是记录你上传博文用的，可以是时间也可以是文本内容。不能为空！！！

git remote add origin https://github.com/hpf189/hpf189.github.io.git
git push -u origin master

以上执行的两个操作将本地的 public 和 github仓库里的关联在public下

然后就可以进入到自己的GitHubpage页面查看更新后的博客啦！
